# Nelhydrogen.com

## Install required javascript libraries

From theme root, run:

```sh
yarn
```

## Build

All scripts are built locally with [CodeKit](https://codekitapp.com/) and pushed to production. Setup is saved in `config.codekit3` located in theme root.
